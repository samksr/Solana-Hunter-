This repository contains a production-ready Solana Hunter Telegram bot with:
- Ultra Dashboard (inline admin panel)
- PumpFun WebSocket sniper
- Raydium pool watcher
- Nitter RSS scanning with failover
- Safe MarkdownV2 escaping and throttled logging

Deploy to Railway or any Node.js host. Set env vars as in .env.example.
